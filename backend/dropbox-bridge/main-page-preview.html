OC Save Keeper Bridge

세이브 백업 앱 OC Save Keeper의 OAuth 브릿지 서비스입니다.

Dropbox가 OAuth Device Code Grant를 공식 지원하지 않아, Switch에서 폴링 기반 인증을 완료하려면 이 브릿지가 필요합니다.

원한다면 직접 호스팅하여 운용할 수 있습니다.

GitHub

설치 방법

  1. Release에서 .nro 파일 다운로드
  2. SD카드 /switch/ 폴더에 복사
  3. Homebrew Menu에서 실행

Dropbox 연동

  1. 앱에서 클라우드 동기화 → Dropbox 로그인 선택
  2. 화면에 표시된 QR코드를 스마트폰으로 스캔
  3. 브라우저에서 Dropbox 계정 로그인 및 권한 승인
  4. 자동으로 인증 완료! 별도 코드 입력 불필요
  5. 클라우드 동기화 활성화 완료!

OC Save Keeper Bridge

OAuth bridge service for OC Save Keeper, a save backup app.

Dropbox does not officially support OAuth Device Code Grant, so this bridge is required for polling-based auth on Switch.

You can also self-host this service.

GitHub

Installation

  1. Download .nro file from Release
  2. Copy to SD card /switch/ folder
  3. Launch from Homebrew Menu

Dropbox Setup

  1. Select Cloud Sync → Dropbox Login in app
  2. Scan QR code with your smartphone
  3. Login to Dropbox and authorize in browser
  4. Auth completes automatically - no code entry needed
  5. Cloud sync activated!
