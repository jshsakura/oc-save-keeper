브릿지 서버 정상 작동 중

OC Save Keeper Bridge

세이브 백업 앱 OC Save Keeper의 OAuth 브릿지 서비스입니다.

GitHub Repository

설치 방법

  1. Release에서 .nro 파일 다운로드
  2. SD카드 /switch/ 폴더에 복사
  3. Homebrew Menu에서 실행

Dropbox 연동

  1. 앱에서 클라우드 동기화 → Dropbox 로그인 선택
  2. 화면에 표시된 QR코드를 스마트폰으로 스캔
  3. 브라우저에서 Dropbox 계정 로그인 및 권한 승인
  4. 인증 완료 화면의 코드를 Switch에 입력
  5. 클라우드 동기화 활성화 완료!

API Endpoints

POST /v1/sessions/start
POST /v1/sessions/{id}/status
POST /v1/sessions/{id}/consume
GET /oauth/dropbox/callback
GET /healthz
