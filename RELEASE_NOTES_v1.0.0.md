첫 정식 릴리즈 — 안정적인 세이브 백업과 클라우드 동기화

## 주요 기능

### 클라우드 동기화 (Dropbox)
- QR 코드 기반 Dropbox 인증 (브리지 서비스 선택사용)
- 계정/기기별 세이브 분리 관리
- 기기 우선순위 기반 충돌 해결
- 클라우드 세이브 목록 화면에서 `X` 버튼으로 즉시 새로고침
- Dropbox 연동 완료 안내 문구 개선

### 로컬 백업
- JKSV 스타일 안전 복원 (자동 롤백 지원)
- ZIP 무결성 검증 후 복원
- `_autosave` 접미사 자동 처리
- 휴지통 기능 (30일 보관)
- 최대 100개 버전 히스토리

### UI/UX
- Catppuccin Mocha 테마
- 한국어/영어 다국어 지원
- 메인 화면 정렬 모드: 이름 / 설치순 / 저장순
- 기본 정렬을 설치순(최신 설치 우선)으로 변경
- 하단 버튼 안내 개선 (`- 정렬` 추가)
- 로딩 팝업으로 긴 목록 조회 시 대기 상태 표시
- 사이드바 40% 너비로 가독성 향상
- 부드러운 스크롤 및 애니메이션

### 보안
- SHA-512 해시 무결성 검증
- SSRF 방어 (URL 화이트리스트)
- ZIP 경로 탐색 공격 방지
- HTTP 요청 리미트

## 포함

- `oc-save-keeper.nro`
- 한국어/영어 UI 문자열
- 닌텐도 스위치 공유 폰트 렌더링
- 메타데이터 기반 클라우드 동기화
- 기기 우선순위 충돌 처리
- 로컬 버전 히스토리 및 복원

## 설치 경로

```
/switch/oc-save-keeper/
├── oc-save-keeper.nro
├── backups/          # 로컬 백업 저장소
├── logs/             # 로그 파일
├── temp/             # 임시 파일
└── config/
    ├── dropbox_auth.json
    └── settings.json  # device_id, device_priority
```

## 사용법

| 버튼 | 메인 화면 | 이력 화면 |
|------|----------|----------|
| A | 열기 / 작업 | 복원 / 다운로드 |
| B | 종료 | 닫기 |
| X | 새로고침 | 새로고침 |
| Y | 언어 전환 | 언어 전환 |
| L | 사용자 변경 | 사용자 변경 |
| R | 연동 설정 | 연동 설정 |
| - | 정렬 모드 | 삭제 |

## 알려진 제한

- 클라우드 버전 브라우징은 단순 날짜 기준으로 상세 조회는 미구현
- 충돌 처리는 자동 결정 + 사유 텍스트 표시 방식
- 드롭박스 연동시 제3자 서버를 경유하는게 불편하다면 직접 호스팅하는 방법밖엔 없음
- 로그인을 위해 기기에 드롭박스 Refresh Token 이 저장되니 기기 양도시 주의 필요 (삭제 권장)

## 테스트 권장

1. 앱 실행 및 폰트 렌더링
2. 타이틀 스캔 및 게임 목록
3. 로컬 백업 생성
4. Dropbox 연동 및 클라우드 업로드
5. 클라우드에서 다운로드 및 복원
6. 삭제 후 휴지통 복원
7. 정렬 모드 전환 (`-` 버튼)
